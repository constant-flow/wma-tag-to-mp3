# WMA tag to MP3 
This tool help to copy media metadata from your `wma`s to `mp3`s near by that don't have these informations

# Converting wma to mp3
The best tool for the conversion job is VLC. There you can set the compression rate you prefer.
Downside, it doesn't copy the media tags. That's why this quick script exists.

# Usage

## Install

- requires `nodejs`
- clone the repo
- `npm install`

## Use

- `npm start`
- Drag from Windows explorer to your terminal (tested with cmder)
- Make sure not to drag to many files, as the drag'n'drop cache is rather small
- confirm with `Enter`
